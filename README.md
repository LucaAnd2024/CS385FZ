# CS385FZ Team Project - MyNote (RN + Python)

## Structure
- doc: project documents (requirements, plan, APIs)
- Mynote_RN: React Native frontend
- MynoteBack: Python backend

## Quick Start

### Frontend

```bash
cd Mynote_RN
npm install
npm run ios
```

### Backend

```
cd MynoteBack
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```
